# 2-fighters
Hello Guys. This is a 2d game made using python library called "pygame".

Anyone can play with their friends as many times as they want (1v1).

Controls :
person 1 
 -> w for jump
 -> a for backward
 -> d for forward
 -> r for attack_type 1
 -> t for attack_type 2

person 2
 -> ↑ for jump
 -> ← for forward
 -> → for backward
 -> 1 for attack_type 1
 -> 2 for attack_type 2

 Enjoy the Game !!!
